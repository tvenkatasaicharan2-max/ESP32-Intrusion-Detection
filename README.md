# AI-Powered Smart Intrusion Detection System (ESP32)
An IoT-based intrusion detection system using PIR, reed switch, and keypad authentication.
Includes Wi-Fi web dashboard and AI-based anomaly detection.

## Hardware
- ESP32
- PIR Sensor (D27)
- Reed Switch (D12)
- Buzzer (D14)
- 4x3 Keypad (D21, D19, D18, D5, D32, D33, D26)

## Features
- Real-time motion and door detection
- Keypad authentication (3 users)
- Lockout after 3 wrong attempts
- Web dashboard (auto-refresh 1s)
- AI anomaly detection on PIR activity
